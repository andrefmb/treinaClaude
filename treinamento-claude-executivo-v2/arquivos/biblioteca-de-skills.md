# Biblioteca de Skills — executivo

Skills por papel. Cada uma tem NOME, DESCRIÇÃO (quando usar) e INSTRUÇÕES.
Crie em Personalizar › Skills › “+ Criar skill”. Todas com a disciplina [VERIFICAR].

## Resumo Executivo de Relatório  _(Todos)_
**Descrição:** Use para transformar um relatório/release em resumo executivo de 1 página.
**Instruções:** Estrutura: (1) números-chave; (2) tendência; (3) os 5 pontos de atenção; (4) prioridades. Cite a origem de cada ponto e liste o que verificar.

## Deck para o Board  _(Estratégia / Diretoria)_
**Descrição:** Use para gerar uma apresentação para o board a partir de um relatório e dados.
**Instruções:** Slides: capa, destaques, tendência (com gráfico), riscos, prioridades, recomendação. Máx. 5 bullets por slide; título = mensagem. Gere .pptx.

## Análise de Concorrência  _(Estratégia / Comercial)_
**Descrição:** Use para comparar concorrentes e ranquear ameaças.
**Instruções:** Tabela: receita, crescimento, margem, canal, força/fraqueza. Ranqueie a ameaça e recomende movimentos. Marque [VERIFICAR] nos números externos.

## Memo de Decisão  _(Diretoria / Estratégia)_
**Descrição:** Use para escrever um memo de decisão de 1-2 páginas ao board.
**Instruções:** Estrutura: situação; opções com prós/contras; recomendação e o que pedimos; riscos. Tom objetivo; [VERIFICAR] nos números externos.

## Brief de Reunião  _(Todos)_
**Descrição:** Use para preparar uma reunião a partir de e-mails e documentos.
**Instruções:** Resumo de 1 página: o que está em jogo, decisões pendentes, e 3 perguntas a fazer. Não envie nada.

## Resposta de E-mail no meu tom  _(Todos)_
**Descrição:** Use para rascunhar respostas de e-mail na minha voz.
**Instruções:** Espelhe o meu tom (do estilo-de-escrita.md). Gere a resposta para eu revisar — nunca envie. Aponte o risco de cada versão se houver.

## Análise de Mercado  _(Estratégia)_
**Descrição:** Use para dimensionar e checar um mercado com pesquisa.
**Instruções:** Tamanho, crescimento, tendências e posição relativa — com fontes. Termine com as perguntas que faltam responder. Trate números como direcionais.
